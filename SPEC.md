# 个人账号密码管理器 - 需求规格文档

## 1. 项目概述

### 项目名称
**密码保险箱** (SafeVault)

### 项目类型
桌面应用程序 - 本地数据存储工具

### 核心功能概述
一款用于安全存储个人账号、密码、邮箱、网址等信息的桌面软件，支持多标签页管理、数据导入导出、本地 JSON 文件存储。

### 目标用户
需要安全管理个人账号信息的普通用户

## 2. UI/UX 规格

### 2.1 窗口设计

#### 主窗口
- **窗口尺寸**: 默认 900x600 像素，最小尺寸 700x500
- **窗口控制**: 最小化、全屏（切换）、关闭、置顶（切换）
- **窗口样式**: 圆角窗口，现代简约风格，带阴影效果
- **标题栏**: 包含应用名称 "密码保险箱" 和窗口控制按钮

#### 置顶功能
- 点击置顶按钮可切换置顶状态
- 置顶激活时，按钮高亮显示
- 置顶状态下窗口始终显示在其他应用上方

## 3. 功能规格

### 3.1 核心功能

#### 标签页管理
- **添加标签**: 点击 "+ 添加标签" 按钮，输入标签名称
- **删除标签**: 悬停标签显示删除按钮，确认后删除（删除前提示）
- **重命名标签**: 双击标签名称可编辑
- **标签排序**: 拖拽调整顺序

#### 数据存储 - 账号密码块
- **数据结构**:
  ```json
  {
    "accountName": "string",
    "username": "string",
    "password": "string",
    "email": "string",
    "note": "string",
    "createdAt": "timestamp",
    "updatedAt": "timestamp"
  }
  ```
- **操作**: 添加、编辑、删除、复制（账号/密码）

#### 数据存储 - 网址块
- **数据结构**:
  ```json
  {
    "name": "string",
    "url": "string",
    "note": "string",
    "createdAt": "timestamp",
    "updatedAt": "timestamp"
  }
  ```
- **操作**: 添加、编辑、删除、打开链接、复制

#### 导入/导出功能
- **导出**: 生成 JSON 文件，文件名包含日期 (safe_vault_20260101.json)
- **导入**: 选择 JSON 文件，解析后合并到当前数据
- **数据验证**: 导入时检查 JSON 格式，提示错误

### 3.2 用户交互

#### 数据编辑
- 点击 "添加" 按钮弹出表单
- 填写完成后点击 "保存"
- 取消按钮关闭表单

#### 复制功能（独立操作，不影响编辑）
- **复制按钮**: 每个数据项右侧有独立的复制图标按钮，点击即可复制对应内容
- **账号密码块**:
  - 独立的账号复制按钮（复制用户名）
  - 独立的密码复制按钮（复制密码）
  - 点击复制按钮不触发编辑模式
- **网址块**:
  - 独立的网址复制按钮
- **复制反馈**: 底部显示 "已复制到剪贴板" 提示，2秒后自动消失
- **快捷复制**: 双击账号/密码/网址的文字可快速复制（区分编辑双击）

#### 搜索功能
- 顶部搜索框，输入关键词实时过滤

### 3.3 数据流

```
用户操作 → 前端处理 → Electron IPC → 主进程 Node.js fs → 文件读写 → 返回结果 → 前端更新
```

- **存储文件**: `safe_vault.json`（存储于 `app.getPath('userData')` 目录，避免权限问题）
- **自动保存**: 数据变更后 1 秒自动保存

## 4. 技术规格

### 4.1 技术栈
- **框架**: Electron 30+ (Node.js + Chromium)
- **前端**: React 18 + Vite
- **IPC**: Electron contextBridge + ipcMain/ipcRenderer
- **打包**: electron-builder，编译为安装包及便携式 .exe

### 4.2 开发环境
- Node.js 18+
- npm 或 pnpm
- 无需 Rust，仅 Node.js 即可完成全部开发

### 4.3 文件结构
```
safe_vault/
├── public/
│   └── index.html
├── src/               # React 前端源码
│   ├── main.jsx       # React 入口
│   ├── App.jsx
│   ├── components/    # UI 组件
│   └── styles/
├── electron/          # Electron 主进程
│   ├── main.js        # 主进程入口（窗口、IPC）
│   ├── preload.js     # contextBridge 预加载脚本
│   └── fileManager.js # 文件读写逻辑
├── package.json
└── vite.config.js
```
