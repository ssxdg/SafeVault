## 1. 软件简介

**密码保险箱**是一款 Windows 本地桌面应用，用于管理个人账号密码、网址 Token 和全局记事本内容。软件完全离线运行，数据保存在本机 JSON 文件中，不需要注册账号，也不会上传到服务器。

### v2.0.0 核心特性

- 多标签分类管理账号密码和网址。
- 新增账号、网址和标签时默认排在前面。
- 复制账号/密码或打开网址后会累计使用次数，并在切换标签或搜索时按使用频率排序。
- 独立全局记事本，支持多个记事本标签、切换、重命名和删除。
- 导入/导出 JSON 备份，导入时自动合并并去重。
- 删除确认、导入导出结果、提示信息统一使用应用内美观弹窗。
- 数据结构升级为 `schemaVersion: 2`，兼容旧版数据。

## 2. 安装与启动

### 安装包安装

1. 双击 `密码保险箱 Setup 2.0.0.exe`。
2. 按安装向导完成安装。
3. 安装完成后可从桌面快捷方式或开始菜单启动。

### 便携版运行

直接双击 `密码保险箱 2.0.0.exe` 即可运行，无需安装。

### 开发环境启动

```bash
npm install
npm run dev
```

如 `7331` 端口被占用，可运行：

```bash
npm run dev:kill
```

### 打包构建

```bash
npm run build
```

构建产物位于 `release/` 目录。

## 4. 标签页管理

标签页用于给账号和网址分类，例如：工作、个人、社交、金融、服务器等。

### 添加标签

1. 点击左侧底部 **+ 添加标签**。
2. 输入标签名称。
3. 按回车确认，或点击输入框外完成。

新标签会自动排在标签列表前面。

### 重命名标签

双击标签名称，输入新名称后按回车确认。

### 删除标签

1. 鼠标悬停在标签上。
2. 点击右侧删除按钮。
3. 在确认弹窗中点击 **删除**。

> 删除标签会同时删除该标签下的账号密码和网址，无法恢复。

### 调整顺序

按住标签拖拽，可以调整标签顺序。

## 5. 账号密码管理

### 字段说明

| 字段 | 说明 |
|------|------|
| 账户名称 | 用于识别账号，例如 GitHub、Gmail、微信 |
| 账号 / 用户名 | 登录账号、手机号或用户名 |
| 密码 | 登录密码，卡片中默认隐藏 |
| 邮箱 | 关联邮箱 |
| 登录网址 | 该账号对应的登录页面 |
| 备注 | 其他说明 |

### 添加账号

1. 选择左侧目标标签。
2. 点击右上角 **+ 账号密码**。
3. 填写表单。
4. 点击 **保存**。

### 编辑账号

点击账号卡片右上角编辑按钮，修改后保存。

### 复制账号、密码、邮箱和网址

- 点击复制按钮可复制对应字段。
- 双击部分文本也可以复制。
- 复制后底部状态栏会显示提示。
- 复制账号/密码会增加该卡片的使用次数。

### 显示/隐藏密码

点击密码旁边的显示按钮，可以在明文和隐藏状态之间切换。

### 删除账号

点击删除按钮，在应用内确认弹窗中点击 **删除**。

### 使用频率排序

软件会记录账号卡片的使用次数。为了避免点击后卡片位置突然跳动，排序只会在以下场景刷新：

- 切换左侧标签。
- 修改搜索关键词。

## 6. 网址管理

### 字段说明

| 字段 | 说明 |
|------|------|
| 网站名称 | 网站或服务名称 |
| 网址 | 完整 URL 地址 |
| Token / Key | API Token、Access Key 等 |
| 备注 | 其他说明 |

### 添加网址

1. 选择目标标签。
2. 点击右上角 **+ 网址**。
3. 填写表单并保存。

### 打开网址

点击网址文本或打开按钮，会使用系统默认浏览器打开网址，同时增加使用次数。

### 复制网址或 Token

点击复制按钮即可复制对应内容。

### 删除网址

点击删除按钮，在应用内确认弹窗中点击 **删除**。

## 7. 全局记事本

v2.0.0 新增独立的全局记事本功能。记事本不属于任何密码标签，不受左侧分类标签影响。

### 进入记事本

点击左侧顶部固定入口 **记事本**。

### 新建记事本

点击记事本顶部标签栏中的 **+** 按钮，会新增一个记事本标签。

### 切换记事本

点击顶部记事本标签即可切换。

### 重命名记事本

双击记事本标签名称，输入新名称后按回车确认。

### 删除记事本

点击记事本标签上的关闭按钮，在确认弹窗中点击 **删除**。

> 软件至少会保留一个记事本。如果只剩一个记事本，删除时会出现提示。

### 编辑和保存

直接在文本区输入内容即可。软件会自动保存：

- 输入停止后自动保存。
- 切换记事本前会先保存当前内容。
- 新建或删除记事本前会先保存当前内容。

### 缩进

在记事本编辑区按 `Tab` 键，会插入两个空格，方便整理文本。

## 8. 搜索与视图切换

### 搜索

在内容区顶部搜索框输入关键词，可过滤当前标签下的账号和网址。

搜索范围：

- 账号：账户名称、用户名、邮箱、备注。
- 网址：网站名称、网址、备注。

### 视图切换

| 按钮 | 模式 | 说明 |
|------|------|------|
| 卡片 | 卡片视图 | 适合查看详情和少量数据 |
| 列表 | 表格视图 | 适合大量账号的快速浏览 |

列表视图同样支持复制、编辑、删除。

## 9. 导入与导出

### 导出数据

1. 点击底部栏 **导出**。
2. 选择保存位置。
3. 导出成功后会出现应用内成功弹窗。

导出文件为 JSON 格式，默认名称类似：

```text
safe_vault_20260511.json
```

### 导入数据

1. 点击底部栏 **导入**。
2. 选择 JSON 备份文件。
3. 软件自动合并数据并显示导入结果。

### 导入去重规则

| 类型 | 去重依据 |
|------|----------|
| 账号密码 | 账户名称或用户名 |
| 网址 | 网站名称 |
| 记事本 | 记事本名称 + 内容 |

导入结果弹窗会显示：

- 新增账号数量。
- 新增网址数量。
- 新增记事本数量。
- 跳过的重复数据数量。

> 建议导入前先导出一次当前数据作为备份。

## 10. 弹窗与提示

v2.0.0 已统一应用内弹窗样式，替代浏览器原生确认框和系统消息框。

### 弹窗类型

- 成功提示：导入成功、导出成功。
- 错误提示：导入失败、导出失败。
- 信息提示：无法删除最后一个记事本等。
- 确认弹窗：删除标签、删除账号、删除网址、删除记事本。

### 操作方式

- 点击 **确定** 完成提示。
- 点击 **取消** 放弃危险操作。
- 点击弹窗外部可关闭普通提示。

## 12. 数据存储与安全

### 存储位置

数据保存在：

```text
C:\Users\<用户名>\AppData\Roaming\safe-vault\safe_vault.json
```

### 数据结构

v2.0.0 使用新版结构：

```json
{
  "schemaVersion": 2,
  "tabs": [],
  "notepads": [],
  "activeNotepadId": ""
}
```

### 自动保存

数据变更后约 1 秒自动保存。记事本输入也会自动保存。

### 写入保护

保存时会先写入临时文件：

```text
safe_vault.json.tmp
```

写入成功后再替换主文件，降低写入过程中异常退出导致数据损坏的风险。

### 安全建议

- 数据文件是明文 JSON，请不要发送给他人。
- 为 Windows 账户设置登录密码。
- 离开电脑时锁屏。
- 定期导出备份。
- 备份文件建议存放在安全位置。

## 13. 常见问题

### Q: 旧版本数据还能用吗？

可以。v2.0.0 会自动兼容旧数据。旧版单一记事本字段 `notes` 会转换为新版多记事本列表中的一个默认记事本。

### Q: 为什么点击复制后卡片没有马上移动？

这是刻意优化。使用次数会立即记录，但排序只在切换标签或搜索时刷新，避免大量卡片时点击后位置跳动，影响查找。

### Q: 如何备份数据？

点击底部栏 **导出**，保存 JSON 文件即可。

### Q: 如何迁移到新电脑？

1. 旧电脑导出 JSON 文件。
2. 新电脑安装密码保险箱。
3. 在新电脑点击 **导入**，选择备份文件。

### Q: 导入后为什么有些内容被跳过？

被跳过的内容通常是系统判断为已存在的重复数据。导入结果弹窗会显示跳过数量。

### Q: 软件是否联网？

不联网。密码保险箱是本地离线应用。

### Q: 数据是否加密？

当前未加密，数据以明文 JSON 保存在本机。请妥善保护系统账户和备份文件。
