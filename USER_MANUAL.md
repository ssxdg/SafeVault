## 1. 软件简介

**密码保险箱**是一款 Windows 桌面应用程序，用于安全管理个人账号、密码、邮箱、网址等信息。所有数据保存在本地，无需联网、无需注册账户。

### 核心特性

- **多标签页管理**：按类别（工作、个人、社交等）分组管理账号和网址
- **一键复制**：点击按钮即可复制账号、密码、网址等信息
- **智能排序**：常用卡片自动靠前排列，方便快速查找
- **数据导入/导出**：支持 JSON 格式的数据备份与迁移
- **本地存储**：数据仅保存在本机，不上传任何服务器
- **窗口置顶**：支持将窗口固定在最前，方便对照填写

## 2. 安装与启动

### 方式一：安装包安装（推荐）

1. 双击 `密码保险箱 Setup x.x.x.exe`
2. 按向导提示完成安装
3. 安装完成后自动启动，同时在桌面和开始菜单创建快捷方式

### 方式二：便携版

直接双击 `密码保险箱 x.x.x.exe` 即可运行，无需安装。

### 卸载

通过 Windows 设置 → 应用 → 找到"密码保险箱" → 卸载。卸载时默认**保留**数据文件，如需彻底清除请手动删除数据目录（见 [数据存储](#11-数据存储与安全)）。

## 4. 标签页管理

标签页用于将账号和网址按类别分组，例如"工作"、"个人"、"社交"、"金融"等。

### 添加标签

1. 点击侧边栏底部的 **"+ 添加标签"** 按钮
2. 输入标签名称，按回车确认

新标签会自动排在最前面。

### 重命名标签

**双击**标签名称，进入编辑状态，修改后按回车确认。

### 删除标签

鼠标悬停在标签上，点击右侧出现的 **🗑️** 按钮，确认后删除。删除标签会同时删除该标签下的所有账号和网址。

### 调整顺序

**拖拽**标签可调整排列顺序。

## 5. 账号密码管理

每个账号卡片包含以下信息：

| 字段 | 说明 |
|------|------|
| 账户名称 | 用于识别该账号的名称（如"Google"、"GitHub"） |
| 账号 | 用户名/手机号/邮箱等登录账号 |
| 密码 | 登录密码（默认隐藏，可切换显示） |
| 邮箱 | 关联邮箱地址 |
| 登录网址 | 该账号的登录页面地址 |
| 备注 | 额外说明信息 |

### 添加账号

1. 选中目标标签页
2. 点击内容区右上角 **"+ 账号密码"** 按钮
3. 在弹出的表单中填写信息
4. 点击 **"保存"**

新添加的账号卡片会出现在最前面。

### 编辑账号

点击卡片上的 **✏️ 编辑** 按钮，修改信息后保存。

### 复制账号/密码

每个卡片上有多组复制按钮：

- **📋 账号**：复制用户名
- **🔑 密码**：复制密码
- **📧 邮箱**：复制邮箱地址
- **🔗 登录网址**：复制登录网址

点击后底部会显示"已复制XX到剪贴板"提示。

### 显示/隐藏密码

点击密码旁的 **👁️** 图标可切换密码的显示与隐藏。

### 删除账号

点击 **🗑️ 删除** 按钮，确认后删除。

### 智能排序

卡片按**使用频率**自动排序——经常复制或打开的卡片会排在前列。排序在切换标签页或搜索时自动更新，操作过程中卡片位置不会跳动。

## 6. 网址管理

每个网址卡片包含以下信息：

| 字段 | 说明 |
|------|------|
| 网站名称 | 网站的名称标识 |
| 网址 | 完整的 URL 地址 |
| Token/密钥 | 与该网站相关的令牌或密钥 |
| 备注 | 额外说明信息 |

### 添加网址

1. 选中目标标签页
2. 点击内容区右上角 **"+ 网址"** 按钮
3. 填写信息后保存

### 打开网址

点击卡片上的 **🌐 打开** 按钮，将在系统默认浏览器中打开该网址。

### 复制网址/Token

点击对应的复制按钮即可。

## 7. 搜索功能

在内容区顶部的搜索框中输入关键词，可实时过滤当前标签下的所有账号和网址。

**搜索范围**：
- 账号卡片：账户名称、账号、邮箱、备注
- 网址卡片：网站名称、网址、备注

搜索不区分大小写。清空搜索框即可恢复显示全部内容。

## 8. 视图切换

内容区支持两种显示模式，通过搜索框右侧的按钮切换：

| 按钮 | 模式 | 说明 |
|------|------|------|
| 📇 | 卡片视图 | 以卡片网格形式展示，适合少量数据 |
| 📋 | 列表视图 | 以紧凑表格形式展示，适合大量数据 |

列表视图下同样支持复制、编辑、删除等操作。

## 9. 导入与导出

### 导出数据

1. 点击底部栏 **📤 导出** 按钮
2. 选择保存位置
3. 导出成功后会弹出提示

导出文件格式为 JSON，文件名包含导出日期（如 `safe_vault_20260101.json`）。

### 导入数据

1. 点击底部栏 **📥 导入** 按钮
2. 选择之前导出的 JSON 文件
3. 系统自动合并数据，弹出导入结果

**导入规则**：

- 同名标签页会合并内容
- **自动去重**：已存在的账号（按账户名称或用户名匹配）和网址（按网站名称匹配）会被跳过，不会重复添加
- 导入结果弹窗会显示新增数量和跳过的重复数量

> ⚠️ 导入前建议先导出备份当前数据。

## 10. 窗口控制

标题栏右侧提供以下窗口控制按钮：

| 按钮 | 功能 | 说明 |
|------|------|------|
| 📌 | 置顶切换 | 激活后窗口始终显示在其他应用上方，再次点击取消 |
| □ | 最小化 | 将窗口最小化到任务栏 |
| □ | 最大化/还原 | 切换窗口全屏与原始大小 |
| ✕ | 关闭 | 关闭应用程序 |

窗口支持拖拽标题栏移动，也支持拖拽边缘调整大小。最小窗口尺寸为 700×500 像素。

## 11. 数据存储与安全

### 存储位置

所有数据保存在本地 JSON 文件中：

```
C:\Users\<用户名>\AppData\Roaming\safe-vault\safe_vault.json
```

### 自动保存

数据变更后 **1 秒内自动保存**，无需手动操作。关闭窗口或退出程序时数据不会丢失。

### 安全建议

- 数据文件为明文 JSON，建议不要将文件分享给他人
- 定期使用导出功能备份数据
- 卸载软件时数据文件默认保留，如需彻底删除请手动清理上述目录

## 12. 常见问题

### Q: 如何备份数据？

使用导出功能（📤 导出按钮），将 JSON 文件保存到安全位置（如 U 盘、云盘）。

### Q: 换电脑后如何迁移数据？

1. 在原电脑导出数据
2. 将导出的 JSON 文件复制到新电脑
3. 在新电脑安装密码保险箱后使用导入功能

### Q: 导入后出现重复卡片？

v1.0.1 版本已支持自动去重。如果仍有重复，可能是账户名称和用户名都不相同，请手动删除重复卡片。

### Q: 密码安全吗？

数据以明文 JSON 存储在本机。建议：
- 为 Windows 账户设置登录密码
- 不要将数据文件分享给他人
- 离开电脑时锁定屏幕

### Q: 软件需要联网吗？

不需要。密码保险箱是完全离线的本地应用，不会连接任何服务器。

### Q: 如何更新版本？

下载新版本安装包，覆盖安装即可。数据文件不受影响。
